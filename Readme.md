
# retsly-js-sdk

  Retsly JavaScript SDK

## Installation

  Install with [component(1)](https://github.com/component/component):

### Quick Start

  ```html
<script src='http://rets.ly/js/sdk'></script>
  ```

### Build from source

  ```bash
  $ component install Retsly/retsly-js-sdk
  $ component build -o ./ -n retsly-js-sdk
  ```
